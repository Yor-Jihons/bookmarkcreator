BookmarkCreator
----------------------------

バージョン:
  BookmarkCreator 1.0.5

著作者:
  Yor-Jihons

用途:
  定義ファイルから(ブックマークとしての)HTMLファイルを生成する。

できること:
  * -gオプションを使ってHTMLのテンプレートファイルを生成すること
  * 定義ファイルからブックマークとしてのHTMLファイルを生成すること

インストール:
  このzipファイルを解凍すれば完了。

使い方:
  [オンラインヘルプ](https://yor-jroom.com/help/ja/bookmarkcreator.html)を参照してください。
  URL: https://yorroom2.cloudfree.jp/help/ja/bookmarkcreator.html

ライセンス:
  * 本体: MITライセンス

動作環境:
  Windows 7 以降

開発環境:
  Windows 11
  C#.NET (.NET6)
